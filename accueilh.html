Logiciel My Campus

Application pour la gestion de logement d'une universite

Ce logiciel sera géré par l'administration qui se charge des logements dans un campus.Avec cette application on pourra savoir les étudiants qui occupent une chambre dans une pavillon,mais aussi on peut savoir dans quelle pavillon ou dans quelle chambre d'une pavillon X se loge un étudiant E.On peut savoir aussi le nombre d'étudiants codifiés ,le nombre de chambres dans une université,le nombre de chambres dans une pavillon le nombre de pavillons dans une université.
